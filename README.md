This is a a user registration form that validates inputs using JavaScript before submission.
Form Validation Requirements:
Full Name-------->  Must not be empty and contain at least 2 words.
Email Address---->  Must follow a valid email format (e.g. example@domain.com).
Password--------->  At least 8 characters, with one uppercase letter, one number, and one special character.
Confirm Password--> Must match the Password field.
Age---------------> Must be 18 or older.
